# **Отчет по реализации проекта Multi-Agent**

### **Требования**
Перед запуском убедитесь, что у вас установлены:
- **Python 3.9+**
- **pip**
- Доступ к **Mistral API** (API-ключ)

## **Инструкция по запуску**
```bash
pip install -r requirements.txt
python -m venv venv
source venv/bin/activate      # Linux / macOS
venv\Scripts\activate         # Windows
MISTRAL_API_KEY=ваш_api_ключ
python src/run_interview.py
```


---


## **Обзор проекта**
Данный проект представляет собой мультиагентную систему для проведения технических интервью, разработанную в соответствии с техническим заданием. Система реализует архитектуру из двух специализированных агентов, взаимодействующих для проведения адаптивного технического собеседования.

---

## **Соответствие требованиям технического задания**
Система полностью соответствует всем обязательным требованиям:

### **Ролевая специализация**
Реализованы два отдельных агента:
- **InterviewerAgent** — ведет диалог и задает вопросы  
- **ObserverAgent** — анализирует ответы и дает рекомендации  

### **Скрытая рефлексия**
Реализован механизм внутреннего диалога между агентами перед генерацией ответа пользователю. Все «мысли» агентов записываются в лог.

### **Сохранение контекста**
Система запоминает историю диалога, предотвращает повторные вопросы и учитывает предыдущие ответы кандидата.

### **Адаптивность**
Сложность вопросов динамически меняется (**junior / middle / senior**) в зависимости от качества ответов кандидата.

### **Устойчивость к вбросам**
Система распознает галлюцинации и попытки ухода от темы, корректно возвращая беседу в нужное русло.
